# DSA Programs

A Next.js application displaying Data Structures and Algorithms programs with code syntax highlighting and copy functionality.

## Features

- Display all programs from `programs.ts`
- Syntax highlighting for C code
- Copy to clipboard functionality
- Dark/Light theme toggle
- Responsive design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
